### IP地址

##### 网络地址转换NAT
- network address translation

- 在专用网连接到因特网的路由器上安装NAT软件，它至少有一个有效的外部全球IP地址

NAT转换表（双向的转换）


| WAN端-外网端                   | LAN端-内网端   |
| ------------------------------ | -------------- |
| ip地址：端口号                 | ip地址：端口号 |
| 报文发出时，会替换源IP和端口号 |收到报文时，会替换目的IP和端口号             |
|                                |                |
